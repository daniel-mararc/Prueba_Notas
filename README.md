# Prueba_Notas

Tecnologias:

- JavaScript (Para los eventos de la pagina)
- PHP (Para la conexion con la base de datos)
- SQL (Para la base de datos)

Funcionamiento:

La pagina esta creada en php entonces necesita apache y tambien necesita el mysql con la base de datos creada
para que funcione toda la pagina correctamente.

Para crear la base de datos esta el sql en el src.

Para crear notas tienes que darle al boton con el mas abajo a la derecha de la pagina ponerle un titulo y una descripcion obligatoriamente
y darle al boton de guardar.

Luego para cambiar cualquier dato de una nota tienes que darle click a la nota y dentro en el menu tienes la opcion de cambiar el titulo y
la descripcion al igual que actuazlizarla o borrarla.

Para salir de una nota al crearla o actualizarla solo funciona dandole al boton de salir.
